This documents describes rules how to create links for components, transformations, and flows. Always follow those rules.

Example of component url:
https://connection.europe-west3.gcp.keboola.com/admin/projects/1392/components/keboola.ex-telemetry-data/9319900
Template is https://connection.<stack_url>.keboola.com/admin/project/<project_id>/components/<component_id>/<configuration_id>

Example of flow url:
https://connection.europe-west3.gcp.keboola.com/admin/projects/1392/flows/9288074
Template is https://connection.<stack_url>.keboola.com/admin/project/<project_id>/flows/<flow_id>

Example of transformations url:
https://connection.europe-west3.gcp.keboola.com/admin/projects/1392/transformations-v2/keboola.snowflake-transformation/9168033
Template is https://connection.<stack_url>.keboola.com/admin/project/<project_id>/transformations-v2/<component_id>/<configuration_id>

Important rules:
 - stack_url is always same for all links
